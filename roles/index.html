ODG Authority System

System Authorization & Role Hierarchy / 自動化驗證系統

ODG【User】
基礎成員
資格認定
加入伺服器並同意 ODG-EULA 後，由系統自動簽發
授權與限制
  • 解鎖公共區域常規文字交流
  • 論壇基礎閱覽權限（僅可加入反應）
  • 可參與一些基礎的社區活動
  • 不提供任何額外功能（如語音等）
  • 無法參與或發起任何的投票活動
  • 無法于論壇發表或留言任何内容
  • 無法傳輸語音、TTS 及媒體文件
ODG【Verified】
驗證用戶
資格認定
需連結以下任一外部平台帳號
且註冊時間需滿 365 天 (1年)
授權與限制
  • 包含 User 所有可用權益
  • 可以發起或參加社區投票
  • 解鎖 VERIFIED 完整語音通訊
  • 解鎖語音訊息、TTS 及圖片上傳
  • 解鎖部分論壇發佈主題與投稿資格
ODG【Premium】
資深用戶
資格認定
需連結以下任一外部平台帳號
且註冊時間需滿 1095 天 (3年)
授權與限制
  • 包含 Verified 所有可用權益
  • 解鎖 PREMIUM 專屬語音區域
  • 可以參加更具權重的 PREMIUM 投票
  • 解鎖 1 個機密板塊的瀏覽與投稿權限
  • 這些内容或許會在未來改動（增刪查改）
ODG【Developer】
開發者
資格認定
強制連結 GitHub 帳號以完成技術認證
授權與限制
  • 基礎權益同步 Verified 級別
  • 解鎖 DEVELOPER COMMUNITY
  • 獲得向專案所有者申請協作開發之資格
  • 可向專案所有者申請成為項目維護人
  • 可接入自己的 GitHub 項目（需通過審核）
  • 不提供上述功能【之外的】任何額外內容
ODG〘 Booster 〙
加成者 / 頂級貴賓
資格認定
為伺服器進行 Server Boost 自動獲取
由 Discord Inc. 託管，贊助停止即自動回收。
至尊全域授權
  • 無視綁定帳號年份/GitHub 等前置門檻
  • 解鎖所有社群功能 (現在與未來)
  • 解鎖全部語音/開發者/隱藏論壇區域
  • 可參與社區任意投票活動，並擁有一票否決權
  • 無法參與 Founder-Vote（元老院投票）
ODG〘 Founder 〙
元老院
資格認定
為伺服器 Server Boost 連續 365 天後，由系統自動簽發
元老院將會對此計算日期，即從閣下開始贊助的那一天記起
授權與限制
  • 永久解鎖 Founder 的多個專屬區域
  • 永久解鎖絕大部分社群功能 (現在與未來)
  • 可參與權威的 Founder-Vote（元老院投票）
  • 可參與社區任意投票活動，並擁有一票否決權
